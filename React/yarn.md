설치해야하는 yarn 리스트(계속 추가)
=======================

1. material ui 패키지 가져오기
```
  yarn add @material-ui/core @material-ui/icons
```
2. 새 프로젝트 만들기
```
  yarn create react-app 폴더이름
```
3. 라우터 설치
(페이지 이동)

```
  yarn add react-router-dom 
```

4. 스타일컴포넌트 설치
(공통된 컴포넌트를 만들어서 프로젝트 안에서 공유)

```
  yarn create react-app 폴더이름
```

5. 리덕스 패키지 설치

```
  yarn add redux react-redux
```
